# 7장 캐시

자주 쓰이는 문서의 사본을 자동으로 보관하는 HTTP 장치이다.

- 캐시는 불필요한 데이터 전송을 줄여 비용을 절감
- 네트워크 병목을 줄여 대역폭 확보
- 원 서버에 대한 요청을 줄여 부하 감소
- 거리로 인한 지연 시간을 줄여 응답 시간 개선
- 어떻게 캐시가 성능으 ㄹ개선하고 비용을 줄이는가
- 캐시를 어디에 위치시켜야하는가
- 캐시된 사본을 신선하게 유지하는 법
- 캐시의 다른 캐시 혹은 서버와 상호작용
