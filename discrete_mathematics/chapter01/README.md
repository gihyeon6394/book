# chapter 01 이산수학 소개와 이 책의 학습 방법

1. 이산수학이란
    - 이산수학의 정의
      > 컴퓨터는 이산되게 데이터를 표현한다
    - 이산수학 학습의 필요성
      > 컴퓨터의 명령 설계 -> 입력 처리-> 결과 출력 의 모든 과정을 이해하는 기본
2. 이산수학의 주제
    - 컴퓨터에서의 연산과 수의 표현
      > 컴퓨터는 인간과 달리 0, 1로만 데이터 표현
    - 문제 해결을 위한 논리와 증명
      > 문제해결을 위한 방법을 논리적으로 확인해야함
    - 데이터 관리와 집합
      > 데이터를 관리하기 위한 집합을 이해
    - 자료 처리를 위한 행렬과 관계
      > 수집한 데이터의 연관성을 정의하고 분석
    - 명령 정의를 위한 함수
      > 컴퓨터는 함수로 명령을 정의
    - 효과적인 문제 해결과 자료 관리를 위한 그래프와 트리
      > 데이터의 시각화
    - 회로 설계와 부울대수
      > 컴퓨터는 회로로 구성되어있음
    - 결과 예측을 위한 확률
      > 확률을 이용해 문제 발생, 해결방법의 수, 가능성 예측
    - 명령의 효율 측정을 위한 알고리즘
      > 프로그램에 대한 설계도
